# 🍕 Pizza Ordering App

A full-stack MERN Pizza Ordering Application where users can build custom pizzas, place orders, track order status, and admins can manage inventory and orders.

---

# 🚀 Features

## User Features

* User Registration & Login
* JWT Authentication
* Custom Pizza Builder
* Order Placement
* Order Tracking
* Order Confirmation Email
* Payment Flow Simulation

## Admin Features

* Admin Dashboard
* Inventory Management
* Order Status Updates
* Low Stock Email Alerts
* Live Order Monitoring

---

# 🛠 Tech Stack

## Frontend

* React.js
* React Router
* Axios

## Backend

* Node.js
* Express.js

## Database

* MongoDB Atlas

## Authentication

* JWT (JSON Web Token)
* bcryptjs

## Email Service

* Nodemailer

# ⚙️ Installation

## Clone Repository

git clone YOUR_GITHUB_REPO_URL

## Frontend Setup

cd client
npm install
npm run dev

## Backend Setup

cd server
npm install
npm run dev

---

# 🔐 Environment Variables

Create a `.env` file inside server folder.

Example:

PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
EMAIL_USER=your_email
EMAIL_PASS=your_email_password
ADMIN_EMAIL=your_admin_email
